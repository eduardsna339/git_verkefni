#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd   change your working directory
	cd ~	change to user home directory
	ls	list contents of current directory
	pwd	print working directory path
	mkdir	make new directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone	clone a repository to pc
	git log		a log for commit history
	git status	shows alot of usefull information
	git diff	show changes between commits, files
	git checkout	switch to a different existing branch

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2
		--staged is to view the changes staged for the next commit
		c) This form is to view the differences between two objects.

4. 	Hvað er útgáfustýring (e.version control)?
version control is a system that records changes to files over time

5.	Hverjir eru helstu kostir við að nota GIT?
support for non-linear development with parallel branches

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
its best to commit often. Git only takes full responsibility for your data when you commit	

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
working directory contains tracked/untracked or modified files. from there files can be commited to the staging area.
from the staging area files can pushed to an online repository
